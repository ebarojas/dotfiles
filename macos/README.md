# Typical mac setup, some defaults and some brew stuff

## Defaults

Run macos_config.sh to set basic preferences

## Brew
On old machine, get a dump

brew bundle dump --file=Brewfile

On new machine, do:

brew bundle --file=~/path/to/Brewfile --verbose

## Extras that require other things

Docker Desktop: https://docs.docker.com/desktop/setup/install/mac-install/
Photoshop
Cursor : curl https://cursor.com/install -fsS | bash
